# 🚗 Automatic Number Plate Recognition (ANPR)

An Automatic Number Plate Recognition system built using **YOLO, OpenCV, EasyOCR, and Streamlit**.

---

## 📌 Features

- Upload vehicle images
- Detect vehicle number plate
- Crop detected plate region
- Extract plate text using OCR
- Interactive Streamlit web application

---

## 🛠 Technologies Used

- Python
- OpenCV
- YOLO (Ultralytics)
- EasyOCR
- Streamlit
- NumPy
- Matplotlib

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone <your-github-repository-link>
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Application

```bash
streamlit run main.py
```

---

## 🚀 Working

1. Upload a vehicle image
2. The system detects the number plate
3. OCR extracts the plate text
4. Final plate number is displayed on screen
